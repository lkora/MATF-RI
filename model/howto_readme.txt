file.txt needs to contain in order, intgers that represent:

number of vehicles in the left lane of the road down
number of vehicles in the right lane of the road down
number of vehicles in the left lane of the road to the left
number of vehicles in the right lane of the road to the left
number of vehicles in the left lane of the road to the right
number of vehicles in the right lane of the road to the right
number of vehicles in the left lane of the road up
number of vehicles in the right lane of the road up
Seconds to change the WE traffic light 
Seconds to change the NS traffic light


Press load to start and when all vehicles exit press end

What end button does is that is writes down the statistics of the simulation
into results.txt file relative to the directory where the .alp file is stored.
